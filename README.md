# 🚀 Mohamed Sayed Portfolio

A modern and interactive portfolio website built with React and Vite to showcase my journey as a Full-Stack Developer, highlighting my projects, technical skills, certifications, achievements, and professional experiences.

Designed with a strong focus on performance, responsiveness, and user experience, the portfolio provides visitors and recruiters with a clear overview of my work and capabilities.

---

## 🌟 Overview

This portfolio serves as my personal digital space where I present:

- Software Development Projects
- Technical Skills
- Certifications
- Volunteering Activities
- Leadership Experiences
- Professional Achievements
- Contact Information

The goal is to create a professional and engaging platform that reflects both my technical background and personal growth.

---

## ✨ Features

### 🎨 Modern UI Design

A clean and visually appealing interface designed to provide a professional first impression.

### 🌙 Dark & Light Mode

Theme switching support for a comfortable viewing experience.

### ⚡ Smooth Animations

Interactive transitions and motion effects powered by Framer Motion.

### 📱 Fully Responsive

Optimized for desktops, tablets, and mobile devices.

### 💼 Project Showcase

Dedicated section for presenting projects with detailed information and technologies used.

### 🏆 Certifications & Achievements

Highlighting certifications, training experiences, competitions, and accomplishments.

### 📊 Interactive Statistics

Animated counters and visual elements that showcase experience and achievements.

### 📬 Contact Integration

Easy communication through integrated contact functionality powered by EmailJS.

### 🔔 User Feedback System

Interactive notifications and feedback messages using React Toastify.
